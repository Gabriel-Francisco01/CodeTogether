Bem-vindo ao Workshop de Programação Web

Aprenda na prática como desenvolver páginas modernas com HTML e CSS.

O que você vai aprender

HTML Semântico

Estruture suas páginas com as tags corretas para garantir acessibilidade e organização.

CSS e Estilização

Aplique estilos, cores, fontes e layouts para deixar sua página visualmente atraente.

Flexbox e Grid

Domine as ferramentas modernas de layout do CSS para criar interfaces responsivas.

Boas Práticas

Escreva um código limpo, organizado e de fácil manutenção desde o início.

Sobre o Projeto

Este é um projeto colaborativo desenvolvido por alunos do curso de Sistemas de Informação da UFU. O objetivo é simular um fluxo real de desenvolvimento usando GitHub.

Os participantes aprendem na prática como fazer fork de repositórios, resolver issues e abrir pull requests.
